- اختر "Deploy manually"

- **تمام! الموقع شغال!** ✅

### 2. **إضافة أسئلة جديدة**
عندما تجيب بالأسئلة، يمكنك إرسالها بطريقتين:

**أ) نص مباشر:**
```
السؤال: ما هي عاصمة مصر؟
الإجابات:
A) القاهرة (صحيح)
B) الجيزة
C) الإسكندرية
D) أسوان
الشرح: القاهرة هي عاصمة مصر الجديدة وأكبر مدنها
```

**ب) ملف JSON:**
```json
{
  "text": "السؤال هنا",
  "options": [
    { "text": "الخيار 1", "correct": true },
    { "text": "الخيار 2", "correct": false },
    { "text": "الخيار 3", "correct": false },
    { "text": "الخيار 4", "correct": false }
  ],
  "explanation": "الشرح العلمي هنا"
}
```
